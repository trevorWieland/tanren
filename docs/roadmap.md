# Roadmap

- Manual and Hetzner VM provisioners

- Intelligent merge conflict resolution workflow
- Issue intake/backlog management pipeline expansion
- HTTP API for external coordinators
- Extended compliance and audit trail reporting

## Long-Term (Q3-Q4 2026)

- Auto-scaling VM pools and cost optimization controls
- Additional cloud provisioners
- Expanded standards profiles and runtime backends
- IDE-centric coordinator surfaces
