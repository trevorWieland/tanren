## Intent

A `solo-builder` or `team-builder` can reach a first project that is ready for planning and execution so Tanren setup ends in usable work context.

## Preconditions

- First-run setup has started.
- The user can create a new project or connect an existing repository.

## Observable outcomes

- The first project is selected as the active project.
- Tanren shows whether product planning, repository analysis, runtime setup, or credentials still need attention.
- The user can begin discovery, planning, or spec creation from the project.

## Out of scope

- Completing every optional integration before first use.
- Running implementation work before required readiness checks pass.
