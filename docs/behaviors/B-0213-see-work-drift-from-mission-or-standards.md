## Intent

A user can see where work drifted from mission or standards so alignment concerns become visible before they compound.

## Preconditions

- Mission, roadmap, standards, or decision context exists.
- Visible work can be compared to that context.

## Observable outcomes

- Tanren identifies work with unclear, weak, or conflicting alignment to mission or standards.
- Each concern links to the relevant mission goal, standard, decision, or work item.
- The user can route concerns to planning revision, standards revision, follow-up work, or dismissal with rationale.

## Out of scope

- Treating all drift as failure.
- Blocking work solely because an automated summary is uncertain.
